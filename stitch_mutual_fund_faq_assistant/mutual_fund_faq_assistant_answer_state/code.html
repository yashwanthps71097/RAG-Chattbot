What is the exit load for Parag Parikh Flexi Cap Fund?

The Parag Parikh Flexi Cap Fund implements a tiered exit load structure to discourage short-term withdrawals. Units redeemed within 365 days from the date of allotment are subject to a 2.00% exit load. For redemptions occurring between 366 days and 730 days, the exit load is reduced to 1.00%, while no exit load is applicable for units held beyond 2 years.
